WeAccept Payment Gateway for Odoo E-Commerce Store
(Tested Odoo 13)

WeAccept Payment Gateway is available in ODOO to making a payment processing easier

Why ODOO Website WeAccept Payment Gateway?

WeAccept is the best payments platform for your business across all touch points. They manage millions of monthly transactions for their clients. Enable WeAccept merchant to perform multiple payment types for their customers to make online purchase(s).

WeAccept payment gateway support USD and EGP Currency.


Customers/Guest Users Features

  • Accept/Pay payment from anywhere any time.
  • Secure 1-click payment using credit/debit card.
  • Seamless integration to enter card details on payment page.

WeAccept Merchant Features

  • Merchant can login to control panel to track payment status
  • Real-time visualized reports
  • Reports: On-line view transaction report
  • Server-to-server Notification to ensure no missing status update

Odoo Admin Features

  • Administration can modify payment status.
  • Administration can any time switch over from Test environment to Production environment and vise versa.
  • Administration can easily enable/disable WeAccept payment method.

Installation & Configuration

1.) Install Module

Install the WeAccept Payment Acquirer module.

2.) Obtain WeAccept Merchant ID, Payment integration ID and Iframe ID

You have to Signup in weaccept.co. Later you'll receive an welcome email that contains details about your account. You have to login and obtain merchant id, payment integration id and iframe id.

3.) Enter Merchant Details

In Odoo navigate to Invoicing or Account menu » Payment Acquirers » Select WeAccept Payment » Click on Configure/Active button.


Enter your Username, Password, Merchant ID, Payment Integration ID and Iframe ID as well as set your environment preferences.


E-Commerce Payment Workflow

1.) Select WeAccept Payment Method

2.) Enter Payment Details

After it will redirect back to out site
